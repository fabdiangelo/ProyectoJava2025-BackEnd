## Tecnologías Utilizadas

### Backend
- **Lenguaje:** Java.
- **Framework:** Spring Boot.
- **Documentación:** Swagger.

### Frontend
- **Biblioteca/UI:** React. 
- **Estilos:** PrimeReact.

### Base de Datos
- **RDBMS:** PostgreSQL.
- **Proveedor:** SupBase.

- **EMail:** Spring Framework Mail.
- **Pagos:** 
  - PayPal Developer API
  - Mercado Pago Developer API.
- **Videos:** Youtube Developer API.
